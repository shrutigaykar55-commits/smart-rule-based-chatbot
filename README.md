# 🤖 Smart Rule-Based Chatbot (Expert System)

A Python-based conversational chatbot that simulates human-like interaction using **pattern matching**, **intent recognition**, and **rule-based responses**.

This project was developed as an internship project based on mentor specifications to create a console-based chatbot that supports:
- greetings
- help queries
- small talk
- domain-specific knowledge questions
- conversation history logging with timestamps

The chatbot uses **regular expressions (regex)** to identify user intents and respond with predefined rules. It also includes a **small knowledge base** for answering questions related to Python, AI, Machine Learning, and chatbots.

---

## ✨ Features

- ✅ Pattern matching using Regular Expressions
- ✅ Intent recognition
  - Greeting
  - Help
  - Small talk
  - Exit
- ✅ Rule-based response generation
- ✅ Small domain knowledge base
- ✅ Interactive console chatbot
- ✅ Timestamped conversation history
- ✅ Automatic chat logging in text file
